- Sequence of zero or more "characters" 
- Strings are deliminated with double quotations marks and support backslash escaping 
	- "Hi'\\\nEveryone"
# Example
- Which are valid [[JSON]] Strings?
	- James  NO
	- "James\\\n" YES
	- "James\\\_Project\\\_1" NO
	- "" YES
